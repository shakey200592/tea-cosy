# Tea Cosy

Welcome to **Tea Cosy** - a simple yet elegant website for a tea shop, created as a practice project to improve my HTML and CSS skills. This website showcases a fictional tea shop that offers a variety of teas and accessories. The goal of this project was to practice building a responsive, visually appealing, and functional website using only HTML and CSS.

## Project Overview
**Tea Cosy** is designed to be a sleek and modern website for a tea shop. The website consists of multiple sections including a "Hero", "featured teas" and "locations sections". The design focuses on simplicity and usability, with a clean layout and responsive design.

## Features
- **Responsive Design**: The website is fully responsive, making it accessible on various devices, from desktops to mobile phones.
- **Clean Layout**: The layout is designed to be simple and user-friendly, with easy navigation.
- **Visual Appeal**: Use of appealing images and consistent color schemes to enhance the visual experience.
- **Semantic HTML**: Proper use of HTML5 elements to create a well-structured and accessible webpage.

## Technologies Used
- **HTML5**: The backbone of the website, providing structure and content.
- **CSS3**: Used for styling the website, including layout, colors, fonts, and responsive design.
